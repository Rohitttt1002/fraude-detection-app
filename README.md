# Fraud Detection Dashboard

A machine learning dashboard built using Python, Streamlit, and XGBoost to identify suspicious credit card transactions.

## Features
- Fraud detection using XGBoost
- Data preprocessing and cleaning
- Interactive Streamlit dashboard
- Real-time prediction visualization

## Technologies Used
- Python
- Streamlit
- XGBoost
- Pandas
- NumPy
- Scikit-learn
